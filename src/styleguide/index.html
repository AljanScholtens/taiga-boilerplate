How does it work?

Taiga is based on ITCSS – ‘Inverted Triangle CSS'. It involves visualising your entire CSS project as a layered, upside-down triangle. This hierarchical shape represents a model that will help you order your CSS in the most effective, least wasteful way.

Learn more about ITCSS by a talk from Harry Roberts.

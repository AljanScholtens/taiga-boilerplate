Set up Taiga

Taiga Boilerplate is a small HTML/Sass framework for starting a new web project. It is mobile-first with a semantic Sass grid and based on the principles of SMACSS and BEM.

  • Get started with Taiga, check the Readme on Github.
  • Need components? Taiga has a small styleguide.
  • If you like Taiga, please star it on Github.
  • Questions? Mail me at aljan@scholtens.me or tweet to @AljanScholtens.
